# 📱 Social Media Post App (MERN Stack)

A full-stack social media application where users can create posts with images and captions.

---

## 🚀 Tech Stack

### 🖥️ Backend

* Node.js
* Express.js
* MongoDB
* Multer (for file handling)
* ImageKit (for image storage)

### 🎨 Frontend

* React
* Axios (for API calls)
* React Router DOM

---

## ✨ Features

* 📸 Upload images with captions
* 📝 Create and store posts
* 📡 Fetch and display all posts
* ☁️ Image storage using ImageKit
* 🔗 REST API integration

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone <https://github.com/Anudeep651/social-media-post.git>
cd social-media-post
```

---

### 2. Setup Backend

```
cd backend
npm install
```

Create a `.env` file:

```
PORT=
MONGODB_URI=
IMAGEKIT_PRIVATE_KEY=

```

Run backend:

```
npm start
```

---

### 3. Setup Frontend

```
cd frontend
npm install
npm run dev
```

---

## 🌐 API Endpoints

* `GET /posts` → Get all posts
* `POST /create` → Create a new post
